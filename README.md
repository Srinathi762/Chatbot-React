# 🤖 Chatbot - React

A modern chatbot application built using **React** and **Vite**. The project provides an interactive and responsive chat interface with a clean user experience. It demonstrates React fundamentals such as component-based architecture, state management, event handling, and responsive UI design.

## 🚀 Features

* 💬 Interactive chatbot interface
* ⚛️ Built with React and Vite
* 🎨 Clean and responsive UI
* ⚡ Fast performance with Vite
* 🔄 Real-time conversation updates
* 📱 Mobile-friendly design
* 🧩 Modular component structure

## 🛠️ Tech Stack

* React
* Vite
* JavaScript (ES6+)
* CSS3
* HTML5

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Srinathi762/Chatbot.git
```

Move into the project directory:

```bash
cd Chatbot
```

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

## 🏗️ Build for Production

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
